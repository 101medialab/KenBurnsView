KenBurnsView
============

Android ImageViews animated by Ken Burns Effect.

![Demo](https://github.com/gotokatsuya/KenBurnsView/blob/master/demo.gif)


## How to use

```
repositories {
    jcenter()
}

dependencies {
    compile 'com.github.goka.kenburnsview:library:1.0.0'
}
```
